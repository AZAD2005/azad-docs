OCP --> OpensShift Container Platform.

Red Hat OpenShift Container Platform (OCP) is used as a secure, enterprise-grade, and automated Kubernetes platform to build, deploy, and manage containerized applications consistently across hybrid cloud, on-premise, and edge environments.


OCP provides extra tools and features to simplify the deployment, management, and scaling of containerized applications. 

OpenShift Container Platform is a platform for developing and running containerized applications.

To start again: 
	Stop the containers --> remove the container --> remove the docker image.
		$ docker stop dockerContainerID
		$ docker rm dockerContainerID
		$ docker rmi dockerImageID

To start again: 
		open GITBASH in "Dockerfile" location.
		$ docker built -t <dockerImageName_anyName> .   ==> dot is important.
		$ docker images
		$ docker run --name <anyContainerName_your_choice> -p 9001:9001 <dockerImageName>
			$ docker run --name ocp-demo-cntnr-1 -p 9001:9001 ocp-demo:1.0
